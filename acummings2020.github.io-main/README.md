# acummings2020.github.io

Source for [acummings2020.github.io](https://acummings2020.github.io) —
Andrew Cummings' personal site. Built with plain
[Jekyll](https://jekyllrb.com) so GitHub Pages builds it automatically.

## Quick start

- **Change your name, tagline, or social links** → edit `_config.yml`.
- **Add a blog post** → create `_posts/YYYY-MM-DD-title.md` (dashes only!).
- **Attach your CV** → drop `cv.pdf` into `/assets/`.
- **Change site colors** → edit the `:root { ... }` block at the top of
  `css/main.css`.

Full walk-through: see [`SITE-INFO.md`](SITE-INFO.md).

## Local preview (optional)

```bash
gem install bundler jekyll
jekyll serve
# open http://localhost:4000
```
